# Project Memory

## Core
Mobile-first food safety app branded "VenueOS". Clean Clinical palette: #fafbfc/#e8ecf1/#2563eb/#16a34a. Sora headings, Manrope body. Sidebar nav desktop, bottom tabs mobile. UK Food Hygiene Rating focus (3 pillars).
Multi-tenant SaaS: Organisation > Site > User. Email magic link for owners, staff code for staff. Supabase RLS enforced.
